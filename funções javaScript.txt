Ffunções javaScript
1°passo:
  escrever a palavra chave "function"
2°passo:
  dá um nome a função
3°passo:
  escrever parentes (), e dentro deles os
  parametros necessarios para a função.
4°passo:
  delimita o inicio e fim da função com o
  simbolo da chave {}
5°passo:
  entre as chaves será escito o código
  da programação da função. se caso necessário
  será usado a palavra-chave return para retorno
  dos valores necessarios.
